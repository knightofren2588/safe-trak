SafeTrack Release Notes

Current Version: v1.2.1

Admin Controls: Add new release notes
v1.2.1 October 7, 2025
Current
New Features
  • Bug Reporting System: Complete bug tracking system with user reports, admin dashboard, and cloud storage integration
  • User Bug Tracking: Users can track their submitted bugs, see admin responses, and mark replies as read
  • Admin Bug Dashboard: Admin can view, filter, reply to, and delete bug reports with status management
Improvements
  • Tailwind CSS Integration: Converted bug reporting UI from Bootstrap to Tailwind for better visibility and consistency
  • Cloud Storage Integration: Bug reports now sync with cloud storage with local fallback for offline functionality
v1.2.0 October 7, 2025
New Features
  • Admin Ownership Transfer: Admin can now change project ownership with password verification
  • Project Creator Display: Shows who created each project (read-only) in edit modal
  • Audit Trail: All ownership changes are logged with timestamp, reason, and admin info
  • Admin Access Control: Admin can now edit ALL projects, not just their own
Improvements
  • Multi-user assignment now properly displays when editing projects
  • Enhanced permission system for admin
  • Improved user selection interface with removable chips
Bug Fixes
  • Fixed project counter to only count assigned projects, not created projects
  • Fixed JavaScript variable scope issue in error handling
  • Fixed assigned users not populating correctly in edit modal
v1.1.0 October 3, 2025
New Features
  • Notes System: Added User Notes and Developer Notes for each project
  • Supabase Integration: All notes stored in cloud database
  • Real-time Sync: Notes sync across all users instantly
Bug Fixes
  • Fixed localStorage conflicts with Supabase storage
  • Resolved note deletion issues
v1.0.0 September 15, 2025
Initial Release
Initial Features
  • Project management with full CRUD operations
  • User management and role-based access
  • Safety dashboard with real-time statistics
  • Multi-user project assignments
  • Export to Excel, PDF, and CSV
  • Screenshot attachment support
  • Custom categories and departments
  • Daily safety quotes
